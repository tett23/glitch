h1. Glitch

いろいろグリッチする。

使い方
<pre>
class SomeObject
  include Glitch
end
SomeObject.new().to\_glitch
</pre>

この場合、self.#to\_sに対してグリッチしたものを返す

<pre>
class Array
  include Glitch

  def setup\_glitch
    self.join()
  end
end
(0..10000).inject([]) do |a, i|
  a << ('a'..'z').to\_a.sample
end.to\_glitch
</pre>

のように、setup\_glitchが定義されていると、setup_glitchの返り値に対してグリッチを実行する
